# Chained

## A low fee, peer to peer money transfer application backed by the blockchain ledger.

### User
- pk
- First Name
- Last Name
- Username
- Photo
- Finacial Information
  - Credit Card 
    - Credit Card Number
    - Expiration Date
    - CVV
    - Card holders Name
  - Checking account
    - Routing number
    - Account number

### Transactions
- Sender
- Recipient
- Amount
- Currency
- Description 
- Time stamp
